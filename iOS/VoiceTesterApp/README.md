# Voice API Tester

A SwiftUI app demonstrating the [xAI Voice API](https://docs.x.ai/docs/guides/tts) — text-to-speech, streaming TTS, and real-time voice agents on iOS and macOS.

## Getting Started

1. Open `VoiceTesterApp.xcodeproj` in Xcode 16+
2. Add your [xAI API key](https://console.x.ai/) in-app or via the `XAI_API_KEY` environment variable in the Xcode scheme
3. Run on a simulator or device (iOS 17+ / macOS 14+)
